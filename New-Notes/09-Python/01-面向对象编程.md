## 1. 编程思想对比

| 对比项 | 面向过程 | 面向对象 |
|--------|----------|----------|
| 视角 | 步骤过程 | 操作对象 |

## 2. 类与对象

### 类（Class）

```python
class Car:
    def __init__(self, brand, color):
        self.brand = brand
        self.color = color
    
    def run(self):
        print(f"{self.brand}汽车跑起来了")
```

### 对象（Object）

```python
# 创建对象
car1 = Car("宝马", "白色")
car2 = Car("奔驰", "黑色")

# 调用方法
car1.run()  # 宝马汽车跑起来了
```

## 3. 三大特性

### 封装

```python
class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # 私有属性
    
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
    
    def get_balance(self):
        return self.__balance
```

### 继承

```python
class Animal:
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return "汪汪汪"

class Cat(Animal):
    def speak(self):
        return "喵喵喵"
```

### 多态

```python
def animal_sound(animal):
    print(animal.speak())

animal_sound(Dog())  # 汪汪汪
animal_sound(Cat())  # 喵喵喵
```
