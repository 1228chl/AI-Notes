# 迭代器与生成器

**标签：** #Python #迭代器

---

## 1. 迭代器

迭代器是一个可以记住遍历位置的对象，实现了`__iter__`和`__next__`方法。

```python
class CountDown:
    def __init__(self, start):
        self.start = start
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.start <= 0:
            raise StopIteration
        self.start -= 1
        return self.start + 1

for num in CountDown(5):
    print(num)  # 5, 4, 3, 2, 1
```

---

## 2. 生成器

生成器是一种特殊的迭代器，使用`yield`关键字返回值。

### 生成器函数

```python
def countdown(n):
    while n > 0:
        yield n
        n -= 1

for num in countdown(5):
    print(num)  # 5, 4, 3, 2, 1
```

### 生成器表达式

```python
# 列表推导式（立即创建所有元素）
squares_list = [x**2 for x in range(10)]

# 生成器表达式（惰性计算）
squares_gen = (x**2 for x in range(10))

print(next(squares_gen))  # 0
print(next(squares_gen))  # 1
```

---

## 3. 生成器的优势

- **节省内存**：不需要一次性加载所有数据
- **惰性计算**：按需生成值
- **可以表示无限序列**
