## 卡片格式说明

每张卡片包含以下部分：

1. **一句话严谨定义**：用一句极其严密的长难句给出定义
2. **核心机制与底层原理**：300-500字深度拆解数学原理和底层逻辑
3. **适用场景与边界条件**：什么时候用、什么时候不用
4. **深度案例拆解**：正例（为什么对）和反例（为什么错）
5. **关联知识网络**：使用Obsidian反向链接连接相关概念
6. **面试/实战深挖指南**：初级和高级追问的回答要点

## 使用建议

1. **学习顺序**：按照编号顺序学习，每个概念都依赖前面的基础知识
2. **关联阅读**：使用反向链接跳转到相关概念，建立知识网络
3. **面试准备**：重点阅读每个卡片的"面试/实战深挖指南"部分
4. **实践应用**：结合"深度案例拆解"部分，在实际项目中应用所学知识
