# 词嵌入技术

**标签：** #NLP #词向量

---

## 1. 什么是词嵌入

词嵌入（Word Embedding）是将离散的词语（或字符）转换为连续的低维向量（词向量），使得神经网络能够理解和处理这些词汇的语义信息。

**为什么需要词嵌入？**

| 传统表示方法 | 问题 |
|--------------|------|
| **One-hot编码** | 高维稀疏，无法反映语义相似性 |
| **词袋模型** | 忽略词序，丢失语义信息 |

---

## 2. Word2Vec

Word2Vec是Google在2013年发布的词向量模型，通过神经网络将单词映射为低维稠密向量。

### 两种训练架构

| 架构 | 核心思想 | 特点 |
|------|----------|------|
| **CBOW** | 用上下文词预测中心词 | 训练速度快 |
| **Skip-gram** | 用中心词预测上下文词 | 适合小数据集 |

---

## 3. GloVe

GloVe（Global Vectors for Word Representation）是斯坦福大学提出的词向量模型，基于全局词共现统计。

---

## 4. FastText

FastText是Facebook提出的词向量模型，将每个词表示为其子词（n-gram）的向量之和。

**优点**：
- 可以为未登录词（OOV）生成词向量
- 支持多语言

---

## 5. 词嵌入层实现（PyTorch）

```python
import torch.nn as nn

# 创建词嵌入层
embedding = nn.Embedding(
    num_embeddings=10000,  # 词汇表大小
    embedding_dim=128      # 词向量维度
)

# 输入词索引，输出词向量
word_indices = torch.tensor([1, 2, 3, 4, 5])
word_vectors = embedding(word_indices)  # shape: (5, 128)
```
