# 提示词工程

**标签：** #Agent #Prompt

---

## 1. 提示词工程原则

基于OpenAI官方文档，提炼出5条大原则：

1. 清晰的指令
2. 文本参考
3. 复杂任务拆分简单子任务
4. 给模型"思考"的时间
5. 借助外部工具

---

## 2. 清晰的指令

### 2.1 详细的描述

不要描述得太笼统，尽量多的提供重要的详细信息或上下文。

**示例**：
```
❌ 不好的提示：写一篇文章
✅ 好的提示：请写一篇关于人工智能在医疗领域应用的科普文章，目标读者是高中生，字数约1500字，包含3个实际案例
```

### 2.2 让模型充当某个角色

```python
你是一位资深的Python开发专家，请帮我分析以下代码的问题...
```

### 2.3 使用分隔符

```python
请将以下文本翻译成英文：
"""
人工智能正在改变世界。
"""
```

### 2.4 提供示例

```python
请将以下句子分类为正面或负面情感：

示例1：
输入："这部电影太棒了！"
输出：正面

示例2：
输入："服务态度很差"
输出：负面
```

### 2.5 指定输出长度

```python
请用100字以内总结这篇文章的核心观点
```

---

## 3. 给模型"思考"的时间

### 思维链提问

在例子中展示一步步的推理过程：

```python
请一步一步思考：

问题：一个农场有15只鸡和8只鸭，如果卖掉3只鸡，剩下多少只家禽？

思考过程：
1. 原来有鸡：15只
2. 原来有鸭：8只
3. 卖掉鸡：3只
4. 剩下的鸡：15 - 3 = 12只
5. 剩下的家禽总数：12 + 8 = 20只

答案：20只
```
