# 大模型幻觉问题

**标签：** #Agent #幻觉

---

## 1. 什么是幻觉

大模型的幻觉是指大模型的生成结果中包含了无根据的或错误的内容。

---

## 2. 幻觉类型

### 2.1 事实性幻觉（Factuality）

**表现**：
- 事实不一致（factual inconsistency）
- 捏造事实（fabrication）

**原因**：
- 知识过时
- 数据不准确

### 2.2 忠实性幻觉（Faithfulness）

**表现**：
- 不遵循指令（Instruction inconsistency）
- 不遵循上下文（in-Context inconsistency）
- 回复内容逻辑不一致（Logical inconsistency）

**原因**：
- 指令的模糊性与复杂性
- 推理能力的不足

---

## 3. 解决方案

| 方案 | 说明 |
|------|------|
| **提示词工程** | 通过清晰的指令减少幻觉 |
| **RAG** | 引入外部知识库验证 |
| **模型微调** | 训练模型生成事实性内容 |
| **AI Agent** | 通过工具调用验证信息 |

---

## 4. 未来趋势

构建"**微调过的模型大脑 + RAG知识心脏 + Agent协作四肢**"的超级个体。
