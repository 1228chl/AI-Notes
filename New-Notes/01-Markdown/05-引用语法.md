# Markdown引用语法

**标签：** #Markdown #基础

---

## 1. 基本引用

使用 `>` 标记：

```markdown
> 这是一段引用。
```

---

## 2. 嵌套引用

引用可以嵌套：

```markdown
> 这是一段引用。
> 
> > 这是嵌套引用。
> 
> 回到外层引用。
```

---

## 3. 引用中的其他元素

引用中可以包含其他Markdown元素：

```markdown
> ### 这是引用中的标题
> 
> - 列表项1
> - 列表项2
> 
> **粗体文字**
```

---

## 4. 最佳实践

- 引用内容前后建议加空行
- 嵌套层级不宜过深
- 引用可以包含代码块、列表、标题等元素
