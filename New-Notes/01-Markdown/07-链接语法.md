# Markdown链接语法

**标签：** #Markdown #基础

---

## 1. 行内链接

```markdown
[文本](URL "可选标题")
```

示例：
```markdown
[Google](https://www.google.com "搜索引擎")
```

---

## 2. 参考式链接

将链接定义为引用，便于复用：

```markdown
[文本] [标签]

[标签]: URL "可选标题"
```

示例：
```markdown
我常去 [GitHub] [1] 和 [Stack Overflow] [so]。

[1]: https://github.com
[so]: https://stackoverflow.com "技术问答"
```

---

## 3. 自动链接

使用 `<URL>` 或 `<email@example.com>`：

```markdown
<https://example.com>
<user@example.com>
```

---

## 4. 相对链接

在本地Markdown文件中可使用相对路径：

```markdown
[关于我们](./about.md)
[图片](../images/photo.jpg)
```

---

## 5. 最佳实践

- 链接文本应该有意义，避免使用"点击这里"
- 外部链接建议在新窗口打开
- 相对链接便于本地文件管理
