# Markdown强调语法

**标签：** #Markdown #基础

---

## 1. 斜体

使用 `*` 或 `_` 包裹文字：

```markdown
*hello*    →  *hello*
_hello_    →  *hello*
```

---

## 2. 粗体

使用 `**` 或 `__` 包裹文字：

```markdown
**world**  →  **world**
__world__  →  **world**
```

---

## 3. 粗斜体

使用 `***` 或 `___` 包裹文字：

```markdown
***!!!***  →  ***!!!***
___!!!___  →  ***!!!***
```

---

## 4. 最佳实践

- 建议使用 `*` 和 `**`，避免与下划线混淆
- 特别是在单词中间使用下划线时容易产生歧义
- 粗体和斜体可以嵌套使用
