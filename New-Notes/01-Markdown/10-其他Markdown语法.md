# Markdown其他语法

**标签：** #Markdown #扩展

---

## 1. 分隔线

使用三个或以上的 `-`、`*` 或 `_` 单独成行：

```markdown
---
***
___
```

建议在前后加空行。

---

## 2. 脚注（扩展）

```markdown
这里需要解释一个概念 [^1]。

[^1]: 这是脚注内容，可以多行。
```

---

## 3. 定义列表（扩展）

```markdown
术语
: 定义内容，可多行。
```

---

## 4. 删除线（GFM）

使用两个波浪线 `~~` 包裹：

```markdown
~~删除的内容~~
```

---

## 5. 高亮/标记（扩展）

```markdown
==需要高亮的内容==
```

---

## 6. 目录（TOC）

```markdown
[TOC]
```

---

## 7. 表情符号

- **复制粘贴**：直接插入 😀 🎉 ❤️
- **短码**（GFM、GitHub）：`:smile:` → 😃 `:heart:` → ❤️

---

## 8. 转义字符

使用反斜杠 `\` 转义特殊字符：

```markdown
\* 不斜体 \*  
\# 不是标题  
\[ 不链接 \]
```
