# Markdown标题语法

**标签：** #Markdown #基础

---

## 1. ATX风格标题

使用 `#` 表示标题级别，共6级。`#` 后必须跟一个空格。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

## 2. Setext风格标题

仅支持1、2级标题，使用下划线标记：

```markdown
一级标题
========

二级标题
--------
```

---

## 3. 最佳实践

- 标题前后建议加空行
- 一篇文章通常只有一个一级标题
- 标题层级不要跳级使用（如从##直接到####）
