# Markdown代码语法

**标签：** #Markdown #基础

---

## 1. 行内代码

使用反引号 `` ` `` 包裹：

```markdown
使用 `printf()` 函数输出。
```

**包含反引号的代码**：使用双反引号包裹：

```markdown
`` `code` ``  →  `code`
```

---

## 2. 围栏式代码块（推荐）

使用三个反引号 ` ``` ` 或三个波浪线 `~~~` 包裹，可指定语言实现语法高亮：

````markdown
```python
def hello():
    print("Hello, world!")
```
````

---

## 3. 缩进式代码块

每行缩进4个空格或1个Tab：

```markdown
    def hello():
        print("Hello")
```

---

## 4. 支持的语言

围栏式代码块支持语法高亮的语言包括：
- Python, JavaScript, Java, C/C++
- HTML, CSS, SQL
- Markdown, LaTeX
- Shell/Bash
- 等等

---

## 5. 最佳实践

- 推荐使用围栏式代码块，支持语法高亮
- 始终指定代码语言以获得最佳渲染效果
- 代码块前后建议加空行
