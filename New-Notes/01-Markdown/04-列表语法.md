# Markdown列表语法

**标签：** #Markdown #基础

---

## 1. 无序列表

使用 `-`、`+` 或 `*` 作为标记，后跟空格：

```markdown
- 苹果
- 香蕉
- 橙子
```

---

## 2. 有序列表

使用数字加点 `.`，数字不必连续，但建议从1开始：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

---

## 3. 嵌套列表

在子项前**缩进2或4个空格**（或一个Tab）：

```markdown
- 水果
  - 苹果
    - 红富士
  - 香蕉
- 蔬菜
  1. 胡萝卜
  2. 西兰花
```

---

## 4. 任务列表（GFM扩展）

使用 `- [ ]` 表示未完成，`- [x]` 表示已完成：

```markdown
- [x] 完成作业
- [ ] 阅读书籍
- [ ] 写报告
```

**注意**：方括号内有一个空格，`x` 大小写均可。

---

## 5. 最佳实践

- 无序列表建议统一使用 `-`
- 有序列表建议按顺序编号
- 嵌套层级不宜过深，一般不超过3层
