# Markdown图片语法

**标签：** #Markdown #基础

---

## 1. 基本语法

语法与链接类似，前面加 `!`：

```markdown
![替代文本](图片URL "可选标题")
```

示例：
```markdown
![Markdown Logo](https://markdown-here.com/img/icon256.png "MD logo")
```

---

## 2. 带链接的图片

图片可以包裹在链接中，点击图片跳转：

```markdown
[![替代文本](图片URL)](点击跳转的URL)
```

---

## 3. 调整图片大小

Markdown本身不支持调整图片大小，但可以使用HTML：

```markdown
<img src="图片URL" width="200">

或者使用百分比：
<img src="图片URL" width="50%">
```

---

## 4. 图片标题

使用HTML的title属性：

```markdown
<img src="图片URL" title="图片标题" alt="替代文本">
```

---

## 5. 最佳实践

- 替代文本应该描述图片内容，便于无障碍访问
- 使用相对路径引用本地图片
- 图片文件建议放在统一的图片文件夹中
