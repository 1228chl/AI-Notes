# 机器学习定义

**标签：** #机器学习 #基础

---

## 1. 核心定义

**机器学习**是一种让计算机**从数据中学习规律**，而**不需要为每个步骤明确编程**的方法。

- **传统编程**：输入 **规则 + 数据** → 输出 **答案**
- **机器学习**：输入 **答案 + 数据** → 输出 **规则**

---

## 2. 三个核心要素

| 要素 | 说明 | 比喻 |
|------|------|------|
| **数据（Data）** | 模型的"教材"，没有数据无法学习 | 教材 |
| **模型（Model）** | 用来学习规律的"数学公式"或"网络结构" | 学习方法 |
| **学习算法（Learning Algorithm）** | 根据数据调整模型参数的"方法" | 学习过程 |

---

## 3. AI、ML、DL的关系

```
人工智能（AI）⊃ 机器学习（ML）⊃ 深度学习（DL）
```

| 层次 | 范围 | 特点 | 典型方法 |
|------|------|------|----------|
| 人工智能 | 最广 | 模拟人类智能 | 专家系统、搜索、规划、推理、学习 |
| 机器学习 | 较窄 | 从数据中学习 | 线性回归、决策树、SVM、聚类 |
| 深度学习 | 最窄 | 深层神经网络自动提取特征 | CNN、RNN、Transformer、GAN |

---

## 4. 机器学习的优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 高精度 | 在图像分类、语音识别等任务上超越传统方法 |
| 端到端学习 | 无需繁琐的手工特征工程 |
| 持续进步 | 每年都有新架构和训练技巧出现 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 网络结构复杂 | 超参数多，对初学者不友好 |
| 训练时间长 | 需要GPU/TPU等加速器 |
| 可解释性差 | 模型内部决策过程难以理解 |
