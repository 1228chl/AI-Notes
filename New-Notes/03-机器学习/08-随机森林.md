# 随机森林

**标签：** #随机森林 #集成学习

---

## 1. 基本概念

随机森林是集成学习的一种，通过构建多棵决策树，综合它们的预测结果来提高模型的准确性和稳定性。

**核心思想**：
- **样本随机**：每棵树使用有放回抽样的训练集
- **特征随机**：每次分裂只考虑部分特征

---

## 2. 算法步骤

1. 从原始训练集中通过bootstrap采样生成多个子集
2. 对每个子集训练一棵决策树
3. 每次分裂时随机选择部分特征
4. 最终预测结果由所有树投票（分类）或取平均（回归）决定

---

## 3. 优缺点

| 优点 | 缺点 |
|------|------|
| 不易过拟合 | 模型可解释性差 |
| 对异常值鲁棒 | 训练时间较长 |
| 可处理高维数据 | 内存消耗大 |
| 可评估特征重要性 | 对噪声较多的数据敏感 |

---

## 4. sklearn代码示例

```python
from sklearn.ensemble import RandomForestClassifier

# 训练模型
rf = RandomForestClassifier(n_estimators=100, max_depth=5, random_state=42)
rf.fit(X_train, y_train)

# 预测
accuracy = rf.score(X_test, y_test)
print(f"准确率：{accuracy:.2f}")

# 特征重要性
importances = rf.feature_importances_
print(f"特征重要性：{importances}")
```

---

## 5. 关键参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `n_estimators` | 树的数量 | 100 |
| `max_depth` | 最大深度 | None |
| `max_features` | 每次分裂考虑的特征数 | 'sqrt' |
| `min_samples_split` | 内部节点再划分所需最小样本数 | 2 |
| `min_samples_leaf` | 叶子节点最少样本数 | 1 |
