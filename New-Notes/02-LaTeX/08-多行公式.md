# LaTeX多行公式

**标签：** #LaTeX #公式

---

## 1. align环境（等号对齐）

使用 `&` 指定对齐位置，`\\` 换行：

```latex
$$
\begin{align}
x &= a + b \\
y &= c + d + e + f \\
z &= g + h
\end{align}
$$
```

效果：

$$
\begin{align}
x &= a + b \\
y &= c + d + e + f \\
z &= g + h
\end{align}
$$

若不想自动编号，使用 `\begin{align*} ... \end{align*}`

---

## 2. gather环境（居中对齐）

```latex
$$
\begin{gather}
x = a + b \\
y = c + d
\end{gather}
$$
```

效果：

$$
\begin{gather}
x = a + b \\
y = c + d
\end{gather}
$$

---

## 3. cases环境（分段函数）

```latex
$$
f(x) = 
\begin{cases}
x^2, & \text{if } x \ge 0 \\
-x^2, & \text{if } x < 0
\end{cases}
$$
```

效果：

$$
f(x) = 
\begin{cases}
x^2, & \text{if } x \ge 0 \\
-x^2, & \text{if } x < 0
\end{cases}
$$

---

## 4. 多行公式手动调整间距

使用 `\\[2ex]` 增加行距：

```latex
$$
\begin{aligned}
y &= x^2 + 2x + 1 \\[2ex]
  &= (x+1)^2
\end{aligned}
$$
```

---

## 5. 注意事项

- `align` 环境会自动编号
- `align*` 环境不编号
- `cases` 环境用于分段函数
- 使用 `\text{}` 在数学模式中插入文本
