# LaTeX括号与定界符

**标签：** #LaTeX #符号

---

## 1. 基本括号

| 描述   | 语法      | 示例代码           | 效果             |
| ---- | ------- | -------------- | -------------- |
| 普通括号 | `( )`   | `(x+y)`        | $(x+y)$        |
| 方括号  | `[ ]`   | `[x+y]`        | $[x+y]$        |
| 花括号  | `\{ \}` | `\{x+y\}`      | $\{x+y\}$      |
| 绝对值  | `\vert` | `\vert x\vert` | $\vert x\vert$ |
| 范数   | `\Vert` | `\Vert x\Vert` | $\Vert x\Vert$ |

---

## 2. 取整与取顶

| 描述 | 语法 | 效果 |
|------|------|------|
| 取整 | `\lfloor \rfloor` | $\lfloor x \rfloor$ |
| 取顶 | `\lceil \rceil` | $\lceil x \rceil$ |

---

## 3. 自动调整大小

使用 `\left...\right` 可以让括号自动适应内容大小：

```latex
$\left( \frac{a}{b} \right)$
```

效果： $\left( \frac{a}{b} \right)$

---

## 4. 手动调整大小

使用 `\big`, `\Big`, `\bigg`, `\Bigg` 手动调整：

```latex
$\Big( (x) \Big)$
```

效果： $\Big( (x) \Big)$

---

## 5. 注意事项

- 在LaTeX中，花括号需要转义：`\{` 和 `\}`
- 使用 `\left` 和 `\right` 配对使用
- `\vert` 和 `\Vert` 用于绝对值和范数，避免与表格分隔符混淆
