# 深度学习概述

**标签：** #深度学习 #基础

---

## 1. 什么是深度学习

深度学习是机器学习的一个子集，使用**深层神经网络**（具有多个隐藏层）来学习数据的层次化特征表示。

**核心区别**：
- **传统机器学习**：依赖人工设计特征（Feature Engineering）
- **深度学习**：**自动提取特征**，端到端学习

---

## 2. 深度学习的优缺点

### 优点

| 优点 | 说明 |
|------|------|
| 高精度 | 在图像分类、目标检测、语音识别等任务上超越传统方法 |
| 端到端学习 | 无需繁琐的手工特征工程 |
| 丰富的框架 | TensorFlow、PyTorch、PaddlePaddle等成熟框架 |
| 持续进步 | 每年都有新架构和训练技巧出现 |

### 缺点

| 缺点 | 说明 |
|------|------|
| 网络结构复杂 | 超参数多，对初学者不友好 |
| 训练时间长 | 需要GPU/TPU等加速器 |
| 可解释性差 | 模型内部决策过程难以理解（"黑盒"） |
| 容易过拟合 | 需要大量数据和正则化技巧 |

---

## 3. 主流深度学习框架对比

| 框架 | 出身 | 动态/静态 | 优势场景 | 入门难度 |
|------|------|-----------|----------|----------|
| **PyTorch** | Meta | 动态（默认） | 学术研究、快速原型 | 低 |
| **TensorFlow** | Google | 静态（2.x支持动态） | 工业部署、大规模分布式 | 中 |
| **PaddlePaddle** | 百度 | 动态/静态可切换 | 国内项目、中文支持 | 低 |
| **ONNX** | 多家联合 | 中间格式 | 模型交换、跨框架推理 | 中 |

## 4. 深度学习发展史

| 时间 | 事件 | 影响 |
|------|------|------|
| 1943年 | McCulloch-Pitts神经元模型 | 人工神经网络的起源 |
| 1957年 | 感知器（Perceptron） | 单层神经网络 |
| 1986年 | 反向传播算法 | 多层网络能够有效训练 |
| 2006年 | 深度置信网络（DBN） | 深度学习正式诞生 |
| 2012年 | AlexNet | 深度学习革命 |
| 2016年 | AlphaGo | 人工智能第三次浪潮 |
| 2017年 | Transformer | 为所有现代LLM奠基 |
| 2022年 | ChatGPT | 大模型时代 |
