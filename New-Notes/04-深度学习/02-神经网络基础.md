## 1. 神经元模型

```
输入: x1, x2, ..., xn
权重: w1, w2, ..., wn
偏置: b

加权求和: z = Σ(wi * xi) + b
激活函数: a = f(z)
输出: a
```

---

## 2. 常用激活函数

| 激活函数 | 公式 | 特点 |
|----------|------|------|
| **Sigmoid** | $\sigma(x) = \frac{1}{1+e^{-x}}$ | 输出(0,1)，易梯度消失 |
| **Tanh** | $\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | 输出(-1,1)，零中心化 |
| **ReLU** | $f(x) = \max(0, x)$ | 计算简单，缓解梯度消失 |
| **LeakyReLU** | $f(x) = \max(\alpha x, x)$ | 解决Dead ReLU问题 |
| **GELU** | $f(x) = x \cdot \Phi(x)$ | Transformer常用 |

---

## 3. 损失函数

### 3.1 分类任务

| 损失函数 | 公式 | 适用场景 |
|----------|------|----------|
| **交叉熵** | $L = -\sum y_i \log(\hat{y}_i)$ | 多分类 |
| **二元交叉熵** | $L = -[y\log(\hat{y}) + (1-y)\log(1-\hat{y})]$ | 二分类 |

### 3.2 回归任务

| 损失函数 | 公式 | 特点 |
|----------|------|------|
| **MSE** | $L = \frac{1}{m}\sum(y_i - \hat{y}_i)^2$ | 对异常值敏感 |
| **MAE** | $L = \frac{1}{m}\sum\vert y_i - \hat{y}_i\vert$ | 对异常值鲁棒 |

---

## 4. 优化器

| 优化器 | 特点 |
|--------|------|
| **SGD** | 随机梯度下降，简单但可能震荡 |
| **Adam** | 自适应学习率，最常用 |
| **AdamW** | Adam+权重衰减，Transformer常用 |
| **RMSprop** | 适合处理稀疏梯度 |

## 5. PyTorch构建神经网络

```python
import torch
import torch.nn as nn

# 定义模型
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.fc1 = nn.Linear(784, 128)
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(0.2)
        self.fc2 = nn.Linear(128, 10)
    
    def forward(self, x):
        x = x.view(x.size(0), -1)  # 展平
        x = self.relu(self.fc1(x))
        x = self.dropout(x)
        x = self.fc2(x)
        return x

# 创建模型
model = Net()

# 定义损失函数和优化器
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
```
