# 微积分

**标签：** #数学 #微积分

---

## 1. 导数

### 1.1 定义

$$
f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}
$$

### 1.2 常用导数公式

| 函数 | 导数 |
|------|------|
| $c$ | $0$ |
| $x^n$ | $nx^{n-1}$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |

### 1.3 求导法则

| 法则 | 公式 |
|------|------|
| 和差 | $(f \pm g)' = f' \pm g'$ |
| 积 | $(fg)' = f'g + fg'$ |
| 链式 | $(f(g(x)))' = f'(g(x)) \cdot g'(x)$ |

---

## 2. 偏导数与梯度

### 偏导数

$$
\frac{\partial f}{\partial x_i} = \lim_{\Delta x_i \to 0} \frac{f(x_1, ..., x_i + \Delta x_i, ..., x_n) - f(x_1, ..., x_n)}{\Delta x_i}
$$

### 梯度

$$
\nabla f = \begin{bmatrix} \frac{\partial f}{\partial x_1} \\ \frac{\partial f}{\partial x_2} \\ \vdots \\ \frac{\partial f}{\partial x_n} \end{bmatrix}
$$

**梯度方向**：函数值增加最快的方向
**负梯度方向**：函数值减少最快的方向

## 3. 积分

### 不定积分

$$
\int f(x) dx = F(x) + C
$$

### 定积分

$$
\int_{a}^{b} f(x) dx = F(b) - F(a)
$$

### 常用积分公式

| 函数 | 积分 |
|------|------|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ |
| $e^x$ | $e^x + C$ |
| $\frac{1}{x}$ | $\ln\vert x\vert + C$ |
